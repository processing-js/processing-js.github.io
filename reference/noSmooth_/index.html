Name

noSmooth()
Processing.js Behavior Processing.js always renders in smooth mode. This is a limitation of the HTML5 Canvas element.
Examples 
background(0);
smooth();
ellipse(70, 48, 36, 36);
noSmooth();
ellipse(30, 48, 36, 36);
Description Draws all geometry with jagged (aliased) edges.
Syntax 
noSmooth()
Returns None
Usage Web & Application
