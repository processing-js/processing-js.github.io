Name

!= (inequality)
Examples 
int a = 22;

int b = 23;

if(a != b) {

  println("variable a is not equal to variable b");

}
Description Determines if one expression is not equivalent to another.
Syntax 
value1 != value2
Parameters
value1 int, float, char, byte, boolean, String
value2 int, float, char, byte, boolean, String
Usage Web & Application
Related > (greater than)
< (less than)
>= (greater than or equal to)
<= (less than or equal to)
== (equality)
