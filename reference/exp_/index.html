Name

exp()
Examples 
float v1 = exp(1.0);
println(v1);  // Prints "2.7182817"
Description

Returns Euler's number e (2.71828...) raised to the power of the value parameter.
Syntax 
exp(value)
Parameters
value int or float: the exponent to raise e to
Returns float
