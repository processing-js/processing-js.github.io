Name

int()
Examples 
float f = 65.0;

int i = int(f);

println(f + " : " + i);  // Prints "65.0 : 65"



char c = 'E';

i = int(c);

println(c + " : " + i);  // Prints "E : 69"
Description Converts a primitive datatype, string, or array to its integer representation.
Syntax 
int(value)
Parameters
value float, char, byte, String, float[], char[], byte[], String[]
Returns int
